# Jak zapnout demo verzi

### Postup pro stáhnutí release verze:
Tento [odkaz](https://github.com/sadronoz/bachelors/releases/tag/final) vás odkáže na stránku s demo.exe souborem. Nemusíte stahovat cokoliv jiného až na .exe soubor.

### Postup přes Godot
1. Stáhněte verzi 4.4 Godotu. Jakákoliv jiná verze může projekt rozbít
2. Stáhněte a rozbalte zip z github repositáře
3. Otevřete projekt jako složku v Godotu
4. Spusťte přes tlačítko či přes klávesu F5

# Jak hrát demo verzi
Pohyb WASD. Pohyb je možný po zahrání tóniny. Hráč má tři životy, které když se sníží na nulu, tak hra skončí.
Při pohybu hráč sekne mečem ve směru kurzoru, jenž dokáže zabít nepřátele. Na kraji a uprostřed stěn jsou neviditelná políčka, která fungují jako dveře do dalších místností. Aktuálně se v dalších místnostech neobjevují další nepřátelé, avšak tato funkce funguje jako ukazatel celé úrovně.
Aktuálně neexistuje žádný cíl v této demo verzi, až na ukázku herních mechanik a jejich propojení.
